Library Database

View on GitHub
More Details
Tools: MySQL, Python, Flask, Docker, html, css
Design of a mock Library database system using SQL. A Flask app pulls book and author data from a MySQL database and displays the information to the user via a web application. Users are able to search the database by title, author, or ISBN number. The app also alows the user to create an account and place a hold on a title.

Spanish Readability Classifier

More Details
Tools: Python, PyTorch, scikit-learn, spaCy
For Spanish language educators and learners, it would be beneficial to have a corpus of online Spanish texts that can be organized based on the difficulilty of the text to a new Spanish learner. This project explores the various features that contribute to Spanish text readability and builds a model to predict the readability of a given text.

Word Search Generator

View on GitHub
More Details
Tools: Python, tkinter, reportlab, cv2 (OpenCV), numpy, PIL, pyPDF2
This is a GUI for a word search generator/creation tool. Users can create a list of words and define the size and shape of the puzzle. Then they can either place the words in the puzzle manually or automatically and generate the rest of the letters around the words. Once the word search has been created, it can be exported as a pdf.
